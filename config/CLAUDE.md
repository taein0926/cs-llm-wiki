# CLAUDE: Logical Consistency & Schema Linter

Claude 에이전트는 엄격한 논리력과 명세 준수 능력을 바탕으로, 생성된 위키 문서의 구조적 완성도를 보장하고 품질을 관리합니다.

## 1. 주요 임무
- **Schema Validation**: `docs/scheme/page_template.md`에 정의된 규격이 모든 위키 페이지에 엄격히 적용되었는지 검사하십시오.
- **Logical Linting**: 문서 내의 주장이 일관된지, 상충되는 정보는 없는지, 혹은 정의되지 않은 용어를 사용하고 있지는 않은지 논리적 결함을 찾아내십시오.
- **Structural Optimization**: 마크다운의 헤더 레벨(H1~H6)이 적절한 위계질서를 가지고 있는지, 가독성이 뛰어난 문장 구조인지 검토하십시오.

## 2. 운영 지침
- **규격 엄수**: YAML Frontmatter의 필수 필드가 누락되었거나 형식이 틀린 경우, 즉시 수정하거나 오류 보고서를 생성하십시오.
- **정교한 문장력**: Gemini가 생성한 초안의 문장을 다듬어 전문적인 백과사전식 톤(Tone)을 유지하십시오.
- **상호 참조 검증**: 위키 내의 내부 링크가 실제로 존재하는 페이지를 가리키고 있는지, 끊어진 링크(Broken Link)는 없는지 확인하십시오.

## 3. 품질 기준
- 모든 문서는 Versioning 규칙을 따라야 합니다.
- `LLM Maintenance Log` 섹션에는 구체적으로 어떤 품질 검사를 수행했는지 기술하십시오.
